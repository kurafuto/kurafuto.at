What is Kyubu?

Kyubu is a parser/serializer for Minecraft written in Golang, intended for developers. It's the workhorse for Kurafuto, turning a useless binary stream into meaningful packets of data.

NOTE! Kurafuto - as well as Kyubu - are both a serious work in progress. It's not exactly vaporware, but don't expect to be able to use this in production yet.

Easy to use

  • Open source, MIT licensed.
  • Written in Go.
  • Easy to extend with custom packet definitions at compile-time.
  • Provides a custom tool (protocol_generator) that works with go generate to avoid writing boilerplate.
  • ...

Wraps an existing io.Reader

package main

import (
	"github.com/kurafuto/kyubu/packets"
	"io"
	"log"
)

func main() {
	// Get this however -- net.Dial, etc.
	var r io.Reader

	// func NewParser(r io.Reader, state State, dir PacketDirection) *Parser
	parser := packets.NewParser(r, packets.Login, packets.ClientBound)

	for {
		packet, err := parser.Next()
		if err != nil {
			log.Fatal(err)
		}

		log.Printf("Packet: %#.2x %T\n", packet.Id(), packet)
	}
}
