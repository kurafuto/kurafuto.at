What is Kurafuto?

Kurafuto is a load-balancer and proxy for Minecraft written in Golang. Trying to be the perfect balance between fully-featured and easily scriptable, Kurafuto aims to please all types of server owners.

NOTE! Kurafuto - as well as Kyubu - are both a serious work in progress. It's not exactly vaporware, but don't expect to be able to use this in production yet.

Fully featured

  • Open source, MIT licensed.
  • Written in Go.
  • Uses less resources than alternatives, as it's not running in the JVM.
  • Designed from the ground up with concurrency patterns in mind.
  • ...

Easily scriptable

  • Features an easily extensible Lua scripting engine.
  • Runs like a dream out-of-the-box.
  • Plug and play repository of extensions.
  • Allows you to talk to the backend servers via Plugin Channels.
  • ...

Installation is a breeze

# Fetch the source and build the Kurafuto binary
$ go get github.com/kurafuto/kurafuto
$ cd $GOPATH/src/github.com/kurafuto/kurafuto
$ go build
# Modify the configuration to your liking
$ vim kurafuto.json
# Run Kurafuto!
$ ./kurafuto -config="$(pwd)/kurafuto.json"

Alternatively, simply download one of our pre-compiled (Linux/amd64) binaries from drone.io.
